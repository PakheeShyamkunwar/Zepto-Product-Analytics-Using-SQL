# Zepto-Product-Analytics-Using-SQL

## Project Overview
This project analyzes Zepto's product inventory dataset using MySQL to uncover insights related to pricing, discounts, stock availability, and category performance.

The objective is to demonstrate SQL skills including data cleaning, aggregation, filtering, sorting, grouping, and business-oriented analysis.

## Business Objectives
- Identify highly discounted products
- Analyze product distribution across categories
- Examine stock availability trends
- Find premium products with high MRP
- Discover categories offering the highest discounts
- Generate actionable business insights for inventory management

## Tools & Technologies
- MySQL
- MySQL Workbench
- SQL

## Dataset Information
The dataset contains information about Zepto products, including:
- Column
- Category
- Name
- MRP
- Discount Percentage
- Available Quantity
- Discounted Selling Price
- Weight in Grams
- Out of Stock
- Quantity
  
## Data Cleaning
Performed the following cleaning operations:
- Removed records with invalid pricing information (MRP = 0)
- Checked for missing values
- Verified data consistency
- Standardized stock availability values
  
## Analysis Performed

### Product Distribution Analysis
- Total number of products
- Products per category

### Pricing Analysis
- Most expensive products
- Average MRP by category

### Discount Analysis
- Top discounted products
- Average discount percentage by category

### Inventory Analysis
- In-stock vs out-of-stock products
- Out-of-stock product investigation
  
### Business Insights
- High-value products
- Discount effectiveness analysis
- Inventory optimization opportunities

## Key Insights
- Majority of products are currently in stock.
- Certain categories offer significantly higher discounts.
- A small number of products showed invalid pricing and were removed during cleaning.
- Discount percentages vary considerably across categories.
- Inventory shortages are concentrated within specific product categories.
